# IJA UML app

JavaFX application to create and edit UML class and sequence diagrams. 

## Authors

- Tomáš Martykán
- Dominik Horký

## Building and running

The project uses Maven and Java 11.
To build and run the JAR, use the following commands:

```
mvn clean package
java -jar ./dest/ija-app.jar
```

To generate the documentation:

```
mvn javadoc:javadoc
```
